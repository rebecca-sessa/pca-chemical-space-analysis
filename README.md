# PCA chemical space analysis

This repository contains a basic cheminformatics workflow focused on molecular descriptor analysis and chemical space visualization using Principal Component Analysis (PCA).

## Overview

The notebook explores dimensionality reduction techniques applied to molecular datasets in order to visualize similarities and differences between compounds according to physicochemical properties.

Principal Component Analysis (PCA) is widely used in cheminformatics and computational drug discovery to simplify multidimensional datasets while preserving relevant variance.

## Contents

- Molecular descriptor calculation with RDKit
- Data preprocessing and scaling
- PCA dimensionality reduction
- Chemical space visualization
- Scientific interpretation of molecular property distributions
